## Getting Started

Simple plant ui app that shows the data from static json data.

Run the following command to install with your fav package manager

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

and then run the following command to start the application.

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
